# Pill Select Input Row

A high-end, modern replacement for the default `input_select` row. 

### Key Features
* **Tile-Card Styling**: Seamlessly matches the latest Home Assistant design language.
* **Smart Dropdown**: Collision detection ensures the menu flips upward if it's near the bottom of your screen.
* **Security Lock**: Optional locking feature to prevent accidental changes.
* **Customizable**: Supports custom widths and "Active State" icon coloring.
